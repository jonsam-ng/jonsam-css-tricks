# jonsam-js-tricks

## run

```bash
yarn storybook
yarn story
```

## build

```bash
yarn build
```

## test

```bash
yarn test
yarn test:watch
```

## generate component

```bash
yarn generate my-component
yarn gen my-component
```



## Dependency

- component template: [react-component-library](https://github.com/HarveyD/react-component-library)
